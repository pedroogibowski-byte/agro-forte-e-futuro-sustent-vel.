# agro-forte-e-futuro-sustent-vel. # 🌱 Agro Forte, Futuro Sustentável

## Tema
**Agro Forte, Futuro Sustentável: Equilíbrio entre Produção e Meio Ambiente**

## Objetivo

Este projeto foi desenvolvido com o objetivo de conscientizar sobre a importância da agricultura sustentável, mostrando que é possível aumentar a produção de alimentos sem prejudicar o meio ambiente.

Por meio da programação, o usuário aprende que pequenas atitudes, como economizar água, preservar as matas, cuidar do solo e utilizar tecnologias sustentáveis, fazem diferença para o futuro.

## Como funciona

O projeto apresenta uma fazenda onde o usuário realiza ações relacionadas à produção agrícola.

Cada escolha influencia dois indicadores:

- 🌾 Produção
- 🌎 Sustentabilidade

Boas decisões aumentam ambos os indicadores, enquanto práticas prejudiciais reduzem a qualidade ambiental e a produtividade.

Ao final, o jogador recebe um resultado mostrando se conseguiu manter o equilíbrio entre produção e preservação da natureza.

## Funcionalidades

- Feedback das escolhas realizadas.

## Conclusão

A tecnologia pode ser uma grande aliada da agricultura sustentável. Este projeto demonstra que produzir alimentos e preservar o meio ambiente são objetivos que podem caminhar juntos, garantindo qualidade de vida para as atuais e futuras gerações.
